基本資訊：系統架構與操作總覽

一句話：客人用手機在 LINE 裡預約潛水、付款；老闆用電腦（或手機）在後台管訂單、會員與收款；系統還會自動幫你催繳、提醒、發通知。

🗺️ 整體架構（一個程式、三種人用 + 同一網址自動分流）

同一個網址會依裝置自動切版面：前台官網 / 手機自動跳手機版 /mobile（Google 爬蟲留原網址收錄）；後台 /admin 手機自動跳 /admin/m（8 大卡），電腦看完整版。

🧩 四大區塊與主要頁面

📱
LINE 會員 App給客人：場次月曆報名、潛旅報名、開團許願、我的預約/付款/退款、會員中心（VIP・抵用金）
🛠️
管理後台給老闆：總覽、老闆結帳、訂單管理、會員管理、客服信箱、願望單、場次/潛旅、抵用金、報表、系統設定
🤿
教練端給教練/助教：今日帶團名單 + 勾到場、本期排班、收款核對（金流限管理者）
🌊
公開官網給路人：首頁、日潛場次、課程、價格、潛點、評價、常見問題、安全/保險、線上諮詢

📚 核心資料概念（白話）

  • 會員身分（角色）：分「會員」與工作人員（助教/教練/管理者/IT/老闆）。工作人員可兼多身分、權限累加；只有「管理者以上」能進後台。
  • 訂單：待付款 → 待審核（客人上傳轉帳截圖）→ 已確認 → 活動結束；途中可能取消。每筆都有完整狀態歷史。
  • 場次 / 潛旅：場次＝某天某時段的日潛（含夜潛）；潛旅＝多天旅行團（有訂金、尾款、繳費期限）。
  • 抵用金：店內折抵金。註冊（Email 驗證）送 50、完成首次潛水（到場）再送 100、生日送、VIP 升等也送。
  • 兩個潛水次數別搞混：「海王子累積氣瓶數」＝實際下水、教練勾到場才加，不能自填，VIP 升等只看這個；「個人資料潛水次數」＝會員自填總經驗，僅供參考、不影響 VIP。
  • VIP 等級：依累積氣瓶數自動升 🦐LV1 → 🦞LV2(21) → 🐢LV3(51) → 🪼LV4(101) → 🐋LV5(201)，每升一級自動發獎勵金。

🤖 自動化系統（系統會自動幫你做的事）

  • 💰 催繳付款：日潛下訂後 D+2 提醒、D+7 最後通知、D+10 自動取消；潛旅訂金 D+5 催、尾款出發前 33/30 天提醒、出發前 2 天寄行前手冊。每段只發一次。
  • 🌬️ 天氣取消：風速超標自動取消整場並退費通知。
  • 🎂 生日抵用金、每日訂單日報、願望單自動關閉、舊憑證自動清理（核可滿 30 天自動刪圖、紀錄留存）。
  • 📨 三種通知管道：LINE / Email / 站內，對象自動分流（客人收確認/催繳/退款；老闆收待審/新願望；教練收超賣警示）。
確認線上跑哪一版：看後台左下角版號，或開 /api/healthz。

⚠️ 老闆該知道的注意細節

  1. 每天打烊看「老闆結帳」+ 清掉待審憑證：客人付款後訂單卡在「待審核」，要盡快核可，否則位子一直懸著。
  2. 教練「勾到場」是 VIP 升級的唯一來源：忘了勾，客人氣瓶數不會加、升不了等。
  3. 刪會員用「軟刪除（封存）」就好：可還原、訂單付款都留；「永久刪除」連訂單憑證一起消失、不可逆。「黑名單」＝能登入但不能預約。
  4. 危險操作（系統設定→危險）：兩種重置都不可復原，需打確認字串才執行。
  5. 群發＝發真實訊息給真人：務必先「試送給自己」。

客戶（會員）角度 · 操作說明

客人從加入到變 VIP，全程在 LINE 裡操作，不用另外下載 App。

🧭 客戶完整旅程

1
加 LINE 好友自動建會員（🦐LV1）
2
完成基本資料填 Email 並驗證 → 送 50 元註冊禮金
3
瀏覽行程日潛場次 / 潛旅團
4
下單填個資+潛水經驗、簽同意聲明
5
付款上傳匯款憑證
6
老闆審核核可 → 訂單確認
7
教練到場簽到氣瓶數+N，首單再送 100
8
累積升 VIP自動升等、發獎勵金

📅 三種預約管道

  • 依日期挑場次 — 場次月曆先選日期再選場次。
  • 直接挑場次 — 從場次列表點進報名（含夜潛）。
  • 潛旅團 — 看團詳情（行程/潛點/費用/繳費期限）→ 報名付訂金。
  • 願望單 — 找不到合適日期可自己提想潛的日期，老闆回覆後開場。
日潛場次開始前 2 小時截止，過了會變灰、點不進去。

📝 下單時要填什麼

會帶出既有資料，但每筆訂單都請客人再次確認/更新：

  • 真實姓名、手機、緊急聯絡人（必填）
  • 必填 證照等級（OW/AOW/DM/Instructor）
  • 必填 證照號碼
  • 必填 累計潛水次數（新手填 0 也可）
  • 多人同行：每位潛伴資料（可從常用潛伴一鍵帶入）
  • 同意聲明：須先閱讀取消/安全政策，再手寫簽名才能送出

💳 付款與憑證上傳

  • 已全面取消現場付款，一律事先付款；到「我的預約 → 付款方式選擇」。
  • 三選一：🏦 銀行轉帳（填後 5 碼）／💚 LINE Pay（截圖必填）／📝 其他（須說明）。
  • 截圖自動壓縮 <500KB；送出後變「⏳ 待老闆審核」，核可後確認；被駁回會顯示說明可重傳。

📂 我的預約 / 會員中心

  • 即將進行的預約 = 未來活動；已結束 = 活動結束（有參加）+ 活動取消（含天氣/客戶取消/未付款不成立）。
  • 每筆可查當初簽的同意聲明；日潛結束後可看當天潛水照片（7 天）。
  • 會員中心：VIP 等級、抵用金/補償金餘額（含明細）、海王子累積氣瓶數、通知偏好（LINE/Email）。

⚠️ 對客戶體驗的注意細節

  1. Email 是兩筆獎勵的鑰匙：沒驗證 Email 拿不到 50 元註冊禮金，首單 100 元也要 Email 已驗證才發 — 提醒新客務必收信點驗證。
  2. 兩個「潛水次數」別搞混：自填經驗不會升 VIP；唯一讓 VIP 升級的是教練當天「到場」勾選。
  3. 每次下單都要重填證照號碼與潛水次數：這是刻意設計（讓教練每場拿到最新經驗），客人覺得「明明填過」是正常的、不是壞掉。

老闆角度 · 後台操作說明

桌機開 /admin 即進後台；左下角有版號。紅色數字徽章＝該頁有待處理，看到就進去處理。

🗂️ 後台六大分類（左側欄）

📊
營運／分析總覽、老闆結帳、報表、前台活動、操作說明
📦
訂單／客戶訂單管理、客製開單、願望單、會員管理、抵用金
🏷️
商品／人員日潛場次、潛水旅行（開團的地方）
📨
客服／通知客服信箱、群發通知、訊息模板、通訊紀錄
📣
行銷節慶優惠、業務推廣、最新動態
⚙️
系統／IT（僅老闆/IT）系統設定、操作紀錄

🚩 開團（日潛 / 潛旅）

  • 日潛場次：新增/編輯單日場次，可設夜潛、預設定價；「Dump 一週」一鍵生整週貼文/場次。
  • 潛水旅行：建團填行程、潛點、費用、訂金與繳費期限。規則：訂金截止＝下訂後 7 天、尾款截止＝出發前 30 天。
  • 注意：已有訂單的場次/團不能硬刪，要先處理訂單。

🧾 訂單與付款

  • 付款憑證審核（最常被漏）：客戶上傳截圖後卡在待審核，核可→確認收款；駁回→請客戶重傳。
  • 改訂單狀態：用狀態 chips 篩選；每筆有完整狀態歷史。
  • 退款/退抵用金：兩段式（一方發起→另一方確認→執行），可退現金或轉抵用金。
  • 老闆結帳（對帳）：打烊前主畫面，先點名「到場」再逐筆/批次「結算收款」。

👥 會員與角色

  • 會員管理：篩選 — 會員 / VIP（LV1~LV5+）/ 教練·助教 / 老闆·管理員·IT / 黑名單。
  • 封存 vs 黑名單 vs 永久刪除：🗃️ 封存（推薦，可還原）／⛔ 黑名單（可登入但不能預約）／🗑 永久刪除（全消失、不可逆）。
  • 發抵用金：手動發放/調整、查明細。VIP 升等獎勵是系統自動發（🤖）。
  • 角色制度：老闆與 IT 身分只能由資料庫設定，後台 UI 改不了。

📣 群發通知（請務必小心）

  • 複選對象：會員、教練助教、某場次參加者…可一次勾多個。
  • 自動去重：同一人落在多群組也只發一次。
  • 自動排除：軟刪除 + 黑名單一律不寄。
  • 送出前列收件名單：顯示人數+名單（已去重/排除），核對再送。
  • 訊息模組：常用內容可存，一鍵載入。
鐵律：先按「📨 試送給自己」確認排版/管道，再正式群發 — 發錯收不回。

⚙️ 系統設定（僅老闆/IT）

分頁設定：💰 金額（裝備費率、教練氣瓶優惠價、生日金、風速門檻）、⭐ VIP（級距＋升級獎勵）、早鳥/折扣與抵用金效期、💳 付款（銀行帳號、LINE Pay ID）、🔗 連結（FB/YT/IG，自動附在通知結尾）、📨 發送（自動通知收件人）、🏠 首頁文案、⚠️ 危險（資料重置，需確認字串、不可復原）。

🕵️ 操作紀錄

稽核所有管理動作（誰、何時、做了什麼），含群發歷史與聯絡客戶紀錄。出事或對帳爭議時來查。

延伸：詳細圖解在後台「操作說明」頁 /admin/guide；手機版後台 /admin/m 也能就地審單、回客服、發抵用金。

⚠️ 老闆最該注意的 5 件事

  1. 群發＝發真實訊息給真人：先試送給自己、核對收件名單人數再送。
  2. 付款憑證要即時審：不審，客戶訂單一直卡待審核、名額被佔。
  3. 天氣取消要「勾通知＋退款」：別讓客人白跑。
  4. VIP 看「海王子累積氣瓶數」不是自填經驗：靠教練現場勾到場累積。
  5. 刪會員優先用「封存」：永久刪除連訂單憑證一起消失不可逆。

教練 / 助教角度 · 操作說明

教練/助教用手機 LINE 開「教練端」，現場帶團、勾到場、上傳動態。

🚪 怎麼進「教練/助教端」

  • LINE 開會員 App → 進「個人中心」。
  • 帳號被設「教練」或「助教」身分 → 多出綠框入口卡（教練後台 / 助教後台）。
  • 點進去＝教練端首頁（今日場次），下方可切「本期排班」「動態管理」。
一般會員看不到這張卡，不怕客人誤入。

✅ 今日帶團 + 到場簽到（最重要）

  • 列出今天每場的時間、潛點、夜潛、人數、客人名單（證照/潛伴/備註/可撥電話）。
  • 每位客人有 到場 / 缺席 兩顆鈕。按「到場」→ 該客人「海王子累積氣瓶數 += 該場氣瓶數×人數」並自動推 LINE 通知。
  • 這是 VIP 升等的唯一來源 — 客人自填的潛數不算，每場每位當天都要確實勾。
  • 勾到場若跨過 VIP 門檻，系統自動升等、發獎勵金、推通知（全自動）。

🌬️ 天氣取消 / 📸 媒體上傳

  • 天氣取消（老闆或教練決定）：一次完成 — 取消該場所有訂單 + 發天氣取消通知（LINE/Email/站內）+ 自動退還客人折抵的抵用金。助教不能做天氣取消。
  • 媒體上傳：在「動態管理」貼上已發在 FB/IG/YT/TikTok 的連結、填日期說明、上傳縮圖（建議 1MB 內）→ 發布，客人在「最新動態」就看得到。

🔐 權限差異（重要）

  • 金流/付款頁，教練和助教都看不到。收款核對、退款、抵用金核可只有「管理者/老闆/IT」進得去；硬點會顯示 🔒 限管理者/老闆/IT。
  • 助教：看場次、出團名單、勾到場 — 就這些。
  • 教練：助教能做的 + 「天氣取消 / 取消改場次」。
  • 兩者都沒有金流與後台管理權限。

⚠️ 注意細節

  1. 沒勾到場 = 沒記到：客人氣瓶數不會 +、VIP 不會升、也不收到場通知。當天逐一勾完。
  2. 勾錯難救：到場會即時加潛數並可能觸發升等發錢，確認本人有下水再勾。
  3. 看到「未收 NT$」只是提醒：收款核對是老闆/管理者的事，教練不經手款項。
