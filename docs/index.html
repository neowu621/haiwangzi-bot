🔁 訂單流程

下單 → 等待付款 → 上傳匯款憑證 → 老闆核對確認 → 到場點名 →（到場結束 / 未到取消）。分岔：突然取消活動（天氣，整場）、客戶取消、訂單不成立；退款分支 refunding → refunded。

🚀 部署

push master → Zeabur 自動建置（約 7–9 分鐘）→ 驗 curl https://haiwangzi.xyz/api/healthz 的 version 即線上版本。版本號在 src/lib/version.ts，每次 push 必 bump（同步更新 CHANGELOG / PROGRESS / status / README）。
