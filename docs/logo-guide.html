白邊防範（重要）

LINE 連結預覽白邊有兩個根因：① favicon 有透明圓角 → 被墊白底；② og 是 webp（LINE 不支援）→ fallback 抓 favicon 再墊白。
現行解法：favicon/apple 一律「滿版深藍、無透明角」；og 一律「整張 JPG、無透明」。 兩者都不會產生白邊。
⚠️ 日後若重出 icon，務必保持滿版無透明；og 保持 JPG。

重製參數（工程備忘）

來源 hwz-deepblue-256.webp → 放大 1024 去背抽白魟(L>115) → 尾巴 nib 拉長 ×1.35 → 凹弧收尖 SHARP=3.6 / CURVE=1.9 → 置中、以翅膀寬為準縮放(FIT 0.86，留白砍半)。深魟版：alpha 內填 #0c283a。
