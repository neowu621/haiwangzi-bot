跟現在差在哪

① 多了「分區標題」(灰字)
像 Cloudflare 的 Observe/組建/保護與連線,把群組再分大類,一眼分層。
② 群組展開 = 子項目縮排 + 左導引線
展開後子項目往右縮、左邊一條細線,層次更清楚(像 WAF/Turnstile 那段)。
③ 目前所在頁 = 淺色 highlight
不是整條變色,而是像 Cloudflare 那種低調的淺灰底(圖中「客服信箱」)。
④ chevron 改成 ▸ → 展開轉 90°
跟 Cloudflare 的 > / ∨ 一致。
保留:深色品牌、我的最愛、待處理徽章。要不要改成 Cloudflare 那種白底淺色主題?可以,但會跟現有後台風格差很多,我建議先維持深色。
