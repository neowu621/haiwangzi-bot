海王子 — 訂單流程說明

從客戶下單、付款、到場點名，到取消與退款的完整路徑。狀態名稱對應程式碼 src/lib/booking-status.ts。

權限模型

桌機後台（完整管理：訂單/會員/場次/退款/設定）— 僅 老闆 (boss) 與 IT（admin 同等）。
教練 (coach) / 助教 (assistant) — 主要用 手機後台（/admin/m/attendance）；即使登入桌機，導覽也只開放「到場點名」。
退款為 boss / admin 專屬，教練/助教不可執行。

狀態圖例

待處理 已確認 / 進行中 完成 取消 / 退款
